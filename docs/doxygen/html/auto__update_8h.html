#define UPDATE_COMMAND
Value:
"TMPDWN=$(curl -s "\
"https://api.github.com/repos/kokkonisd/pixhdl/"\
"releases/latest | grep \"tag_name\" | awk '{print "\
"\"https://github.com/kokkonisd/pixhdl/archive/\" "\
"substr($2, 2, length($2)-3) \"/pixhdl-\" "\
"substr($2, 3, length($2)-4) \".tar.gz\"}') && "\
"curl -L -s $TMPDWN -o /tmp/pixhdl_latest.tar.gz && "\
"tar xvzf /tmp/pixhdl_latest.tar.gz -C /tmp/ && "\
"cd /tmp/pixhdl-* && make clean && "\
"sudo make uninstall && make && sudo make install && "\
"cd .. && rm -rf pixhdl-* pixhdl_latest*"

Command to update pixhdl to its latest version.

This command gets the latest version of pixhdl, downloads it in tmp/, then runs:

$ make clean
$ sudo make uninstall
$ make
$ sudo make install

It finally removes the downloaded files, and the installation is complete.

◆ update()

int update ( )

Check for an update and install newer version.

This function compares the current version with the latest version currently available on the official GitHub repo of pixhdl. If a newer version is available, it asks the user for confirmation, then downloads and installs the newer version.

Returns
(int): 1 if successful, 0 otherwise.
