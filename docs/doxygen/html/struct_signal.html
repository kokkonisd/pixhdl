Detailed Description

Definition of the Signal object.

A Signal is simply an input/output/throughput of an Entity. It can be a port (IN, OUT, INOUT) or a generic input (GENERIC). It has a name, a direction and a length (or value, in the case of generic inputs).

Length of the signal (in bits). It's a string because the parser will attempt to calculate its integer value, but will leave it as-is if it fails (for example if it contains generics).
